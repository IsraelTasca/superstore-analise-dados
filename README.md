# Superstore - Análise de Dados

Projeto simples de análise exploratória de dados utilizando o dataset Superstore.
O objetivo é praticar análise de dados com Python e Pandas.

## Ferramentas
- Python
- Pandas
- Google Colab

## O que foi analisado
- Vendas por categoria
- Lucro por categoria
- Margem de lucro
- Impacto dos descontos
- Subcategorias com prejuízo

## Principais resultados
- A categoria Technology apresentou maior faturamento e boa margem
- Office Supplies teve margem estável
- Furniture apresentou alta venda, porém baixa margem
- Tables e Bookcases geraram prejuízo
- Descontos elevados impactaram o lucro da categoria Furniture

## Conclusão
Este projeto mostra que vender mais não significa, necessariamente, lucrar mais.
